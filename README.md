# YouTube Clone

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)

A simple **YouTube homepage clone** built using **HTML and CSS**.  
This project recreates the basic layout of YouTube, including the header, sidebar navigation, and video grid. The goal of the project is to demonstrate fundamental front-end development concepts such as page structure, layout design, and styling using pure HTML and CSS.

## Purpose

This project was built as an introductory front-end project focused on strengthening core web development fundamentals using HTML and CSS. The focus was on recreating a familiar user interface while understanding how layouts and UI components are structured.

Key areas of focus included:

- Structuring web pages using semantic HTML  
- Styling layouts using CSS  
- Creating a video grid similar to YouTube  
- Building a structured interface with a header and sidebar navigation  
- Organizing styles across multiple CSS files for maintainability

## Setup Instructions

Clone the repository:

```bash
git clone https://github.com/mabhishek-dev/youtube-clone.git
cd youtube-clone
```

Then open:

```
index.html
```

in your browser.

Since this is a static project, no dependencies or build tools are required.
